# Newsletter com Alternância de Tema (Light/Dark)
Aplicação web desenvolvida utilizando HTML, CSS e JavaScript que permite a inscrição em uma newsletter, com funcionalidade de alternância entre tema claro e escuro.
## Sobre o Projeto
### Este projeto tem como objetivo aplicar conceitos fundamentais de desenvolvimento web, como:

• Estruturação de páginas com HTML

• Estilização com CSS (incluindo uso de variáveis CSS)

• Manipulação de DOM com JavaScript

• Validação de formulários

• Persistência de dados no navegador (localStorage)

### O sistema permite:

• Alternar entre tema claro e escuro

• Salvar a preferência de tema do usuário

• Preencher formulário de inscrição

• Validar nome (apenas texto)

• Validar email com padrão correto

• Feedback visual para dados inválidos

## Tecnologias utilizadas:

• HTML5

• CSS3

• JavaScript

• LocalStorage

• Persistência do tema utilizando localStorage

• Validação de formulário com HTML5 (pattern e required)

• Validação adicional com JavaScript

• Atualização dinâmica do ano no rodapé

## Exemplo de Uso

• Clique no botão "Alternar tema" para mudar entre modo claro e escuro

• Preencha o formulário com nome e email válido

• Caso os dados estejam incorretos, o sistema exibirá validação
